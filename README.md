# Hyrox Training — Bordeaux

Site de suivi et préparation pour la course **Hyrox Bordeaux** (2 octobre 2026, Parc des Expositions Bordeaux-Lac).

Page HTML autonome (PWA installable), sans dépendance ni build, avec plan d'entraînement hebdomadaire et suivi de séances.

## Utilisation

Ouvrir `index.html` dans un navigateur, ou activer GitHub Pages sur ce dépôt (Settings → Pages → Deploy from branch → `main`) pour y accéder en ligne et l'installer sur l'écran d'accueil (iPhone/Android).

## Stack

- HTML/CSS/JS vanilla, fichier unique
- Manifest PWA pour installation en application
